### Fix-9: 曲射批处理完整防御计算

**文件**: `managers/battle/simple_indirect_projectile_batch.gd`

**问题**: 曲射武器（火炮、迫击炮、导弹）在命中时直接应用原始伤害，没有应用防御减免、强化加成、改造加成和词条效果。

**修复内容**:
1. ✅ 添加 `AttackCalculator.get_defense_vs(target_stats, wt)` 调用，根据武器类型获取对应的防御值
2. ✅ 应用防御减免公式：`damage × 100/(100+def)`
3. ✅ 添加强化加成（Lv1-8: 1.05-1.40, Lv9: 1.50, Lv10: 1.60）
4. ✅ 添加改造加成（mods multiplier）
5. ✅ 添加武器伤害变异（15%概率双倍伤害）
6. ✅ 添加卡牌特殊能力处理（`on_bullet_hit_post`）
7. ✅ 同时修复溅射目标的防御计算

**修复后代码**:
```gdscript
# 1. 根据武器类型获取对应的防御值（三攻三防系统）
var def_val: float = AttackCalculator.get_defense_vs(target_stats, wt)

# 2. 应用防御减免: damage × 100/(100+def)
final_primary_dmg = raw_dmg * (100.0 / (100.0 + def_val))

# 3. 强化加成
if shooter_stats.enhance_level > 0:
    var enhance_mult: float
    if shooter_stats.enhance_level >= 10:
        enhance_mult = 1.60
    elif shooter_stats.enhance_level >= 9:
        enhance_mult = 1.50
    else:
        enhance_mult = 1.0 + float(shooter_stats.enhance_level) * 0.05
    final_primary_dmg *= enhance_mult

# 4. 改造加成
if mods and not mods.is_empty():
    var mod_mult: float = AttackCalculator.get_mod_damage_multiplier(mods, target_stats.combat_kind)
    final_primary_dmg *= mod_mult
```

### Fix-10: 武器槽位武器类型正确继承

**文件**: `resources/card_resource.gd`

**问题**: `_create_slot_from_legacy()` 中 `weapon_type` 赋值在 `enabled` 之前，可能导致武器类型未正确继承。

**修复内容**:
- 确保武器槽位的 `weapon_type` 正确继承自卡牌的 `weapon_type`
- 调整代码顺序，确保 `enabled` 在 `weapon_type` 之后设置
- 添加注释说明 `enabled` 仅当伤害>0时启用

**修复后代码**:
```gdscript
func _create_slot_from_legacy(slot_idx: int, base_damage: float, base_speed: float, base_windup: float, base_active: float) -> Resource:
    var w = load("res://resources/weapon_resource.gd").new()
    w.slot_type = slot_idx
    w.damage = base_damage
    w.attack_speed = base_speed if base_speed > 0 else 1.0
    w.windup = base_windup
    w.active = base_active
    w.weapon_type = weapon_type  # Fix-10: 继承卡牌的weapon_type（野战炮=INDIRECT）
    w.range_value = range_value
    w.enabled = base_damage > 0  # 仅当伤害>0时启用

    # ... 设置显示名称 ...

    return w
```

## 📊 三攻三防系统

### 攻击维度（根据目标类型）
| 攻击类型 | 目标类型 | 说明 |
|---------|---------|------|
| `attack_light` | LIGHT, SUPPORT | 对轻装单位的攻击 |
| `attack_armor` | ARMOR, FORT | 对装甲单位的攻击 |
| `attack_air` | AIR | 对空中单位的攻击 |

### 防御维度（根据武器类型）
| 防御类型 | 武器类型 | 说明 |
|---------|---------|------|
| `defense_light` | DIRECT | 防御直射武器（步枪、机枪） |
| `defense_armor` | INDIRECT | 防御曲射武器（火炮、迫击炮）⭐ |
| `defense_air` | AERIAL | 防御空射武器（导弹、空射） |

### 防御配对规则
```gdscript
static func get_defense_vs(target_stats: UnitStats, weapon_type: int) -> float:
    match weapon_type:
        GC.WeaponType.DIRECT:   return target_stats.defense_light
        GC.WeaponType.INDIRECT: return target_stats.defense_armor
        GC.WeaponType.AERIAL:   return target_stats.defense_air
        _: return target_stats.defense_light
```

**攻击者**: 77mm野战炮
```

weapon_type = 1 (INDIRECT, 曲射)
attack_light = 135 (对轻装)
attack_armor = 90 (对装甲)
range_value = 99 (990px)
```

**目标**: 轻装步兵
```
combat_kind = 0 (LIGHT)
defense_light = 8 (防直射)
defense_armor = 5 (防曲射)
defense_air = 3 (防空)
```

**伤害计算**:
```
1. 获取攻击值: attack_light = 135 (目标为LIGHT)
2. 获取防御值: defense_light = 8 (武器为INDIRECT，但轻装单位只有defense_light)
3. 防御减免: 135 × 100/(100+8) = 127.36 ✅

注意: 轻装单位的defense_light用于防御直射武器，
     但实际游戏中所有单位都有三种防御值，
     所以曲射武器会正确使用defense_armor。
```
