# 音效资源目录

## 需要的音效文件

### 按钮音效
- button.ogg - 按钮点击音效
- button_hover.ogg - 按钮悬停音效

### 战斗音效
- hit.ogg - 命中音效
- shoot.ogg - 射击音效
- explosion.ogg - 爆炸音效
- cast.ogg - 技能释放音效
- hurt.ogg - 受伤音效

### 结果音效
- win.ogg - 胜利音效
- lose.ogg - 失败音效

### 系统音效
- synthesis_success.ogg - 合成成功
- synthesis_fail.ogg - 合成失败
- blueprint_unlock.ogg - 蓝图解锁
- enhance.ogg - 强化成功
- achievement.ogg - 成就解锁
- quest_complete.ogg - 任务完成

### UI音效
- panel_open.ogg - 面板打开
- panel_close.ogg - 面板关闭
- card_pickup.ogg - 卡牌拾取
- card_place.ogg - 卡牌放置
- error.ogg - 错误音效

## 注意事项
- 推荐格式：OGG (较小文件大小)
- 备用格式：WAV
- 采样率：44.1kHz
- 比特率：128-192 kbps
- 单声道即可（立体声也可以）

## 临时解决方案
如果缺少音效文件，AudioManager会静默跳过，不影响游戏运行。
