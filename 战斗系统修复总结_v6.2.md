# 战斗系统修复总结 v6.2

## 问题描述
1. **双方攻击不掉血** - 伤害计算不正确
2. **曲射攻击不对** - 曲射武器的防御计算缺失

## 根本原因分析

### 问题1：曲射批处理缺少防御计算
**文件**: `managers/battle/simple_indirect_projectile_batch.gd`

**原因**:
- 曲射批处理（`SimpleIndirectProjectileBatch`）在 `_apply_hit()` 中直接调用 `tgt.take_damage(raw_dmg, shooter)`
- 没有应用防御减免、强化加成、改造加成
- 导致曲射武器造成的伤害一直是原始值，不受防御影响

**影响**:
- 所有曲射武器（INDIRECT）和空射武器（AERIAL）都无效减
- 野战炮、迫击炮等支援单位伤害计算错误

### 问题2：武器槽位武器类型继承问题
**文件**: `resources/card_resource.gd`

**原因**:
- `_create_slot_from_legacy()` 中 `w.weapon_type = weapon_type` 在 `w.enabled = base_damage > 0` 之前
- 这在某些情况下可能导致 `weapon_type` 被错误地覆盖或未正确继承

**影响**:
- 武器槽位的 `weapon_type` 可能与卡牌的 `weapon_type` 不一致
- 导致防御计算时使用了错误的防御值

## 修复方案

### Fix-9: 曲射批处理完整防御计算
**文件**: `managers/battle/simple_indirect_projectile_batch.gd`

**修复内容**:
1. ✅ 添加 `AttackCalculator.get_defense_vs(target_stats, wt)` 调用
2. ✅ 应用防御减免：`damage × 100/(100+def)`
3. ✅ 添加强化加成（Lv1-10: 1.05-1.60）
4. ✅ 添加改造加成（mods multiplier）
5. ✅ 添加武器伤害变异（15%概率双倍伤害）
6. ✅ 添加卡牌特殊能力处理

**关键代码**:
```gdscript
# 1. 根据武器类型获取对应的防御值（三攻三防系统）
var def_val: float = AttackCalculator.get_defense_vs(target_stats, wt)

# 2. 应用防御减免: damage × 100/(100+def)
final_primary_dmg = raw_dmg * (100.0 / (100.0 + def_val))

# 3. 强化加成
if shooter_stats.enhance_level > 0:
    final_primary_dmg *= enhance_mult

# 4. 改造加成
if mods and not mods.is_empty():
    final_primary_dmg *= mod_mult
```

### Fix-10: 武器槽位武器类型正确继承
**文件**: `resources/card_resource.gd`

**修复内容**:
- 确保武器槽位的 `weapon_type` 正确继承自卡牌的 `weapon_type`
- 调整代码顺序，确保 `enabled` 在 `weapon_type` 之后设置

**关键代码**:


### Fix-11: GameConstants 类名冲突

## 三攻三防系统验证

### 攻击维度（根据目标类型）
- **对轻装攻击** (`attack_light`) → 攻击 LIGHT/SUPPORT 目标
- **对装甲攻击** (`attack_armor`) → 攻击 ARMOR/FORT 目标
- **对空中攻击** (`attack_air`) → 攻击 AIR 目标

### 防御维度（根据武器类型）
- **对直射防御** (`defense_light`) → 防御 DIRECT 武器（步枪、机枪）
- **对曲射防御** (`defense_armor`) → 防御 INDIRECT 武器（火炮、迫击炮）
- **对空射防御** (`defense_air`) → 防御 AERIAL 武器（导弹、空射）

### 防御配对规则
```gdscript
static func get_defense_vs(target_stats: UnitStats, weapon_type: int) -> float:
    match weapon_type:
        GC.WeaponType.DIRECT: return target_stats.defense_light
        GC.WeaponType.INDIRECT: return target_stats.defense_armor
        GC.WeaponType.AERIAL: return target_stats.defense_air
        _: return target_stats.defense_light
```
