# 游戏内快速测试指南

## 启动游戏后，在任意场景的 `_ready()` 中添加以下代码进行快速测试：

### 方法1: 使用验证器类（推荐）

```gdscript
# 在 main.gd 或任意场景的 _ready() 中:
const BattleSystemValidator = preload("res://scripts/battle/battle_system_validator.gd")

func _ready() -> void:
    # 延迟1秒执行，确保所有资源已加载
    await get_tree().process_frame
    await get_tree().process_frame

    print("\n\n========== 战斗系统自动验证 ==========")
    BattleSystemValidator.quick_validate()
    print("========== 验证完成 ==========\n")
```

### 方法2: 手动检查野战炮

```gdscript
func _ready() -> void:
    await get_tree().process_frame

    var DefaultCards = preload("res://data/default_cards.gd")
    var card = DefaultCards.get_card_by_id("ww1_77mm")

    if card == null:
        print("❌ ww1_77mm 卡牌不存在")
        return

    print("\n========== 野战炮数据检查 ==========")
    print("card_id: ", card.card_id)
    print("combat_kind: ", card.combat_kind, " (2=SUPPORT)")
    print("weapon_type: ", card.weapon_type, " (1=INDIRECT)")
    print("range_value: ", card.range_value)
    print("attack_light: ", card.attack_light)
    print("attack_armor: ", card.attack_armor)
    print("attack_air: ", card.attack_air)
    print("defense_light: ", card.defense_light)
    print("defense_armor: ", card.defense_armor)
    print("defense_air: ", card.defense_air)

    # 检查武器槽位
    if card.weapon_slots.is_empty():
        print("\n⚠️  weapon_slots 为空，正在初始化...")
        if card.has_method("initialize_weapon_slots"):
            card.initialize_weapon_slots()
            print("✅ 初始化完成")

    if not card.weapon_slots.is_empty():
        print("\n武器槽位:")
        for i in range(card.weapon_slots.size()):
            var slot = card.weapon_slots[i]
            if slot != null and slot.enabled:
                print("  槽位 %d: damage=%.1f, weapon_type=%d, enabled=%s" % [
                    i, slot.damage, slot.weapon_type, slot.enabled
                ])

    print("========== 检查完成 ==========\n")
```

### 方法3: 测试伤害计算

```gdscript
func _ready() -> void:
    await get_tree().process_frame

    var UnitStats = load("res://resources/unit_stats.gd")
    var AttackCalculator = preload("res://scripts/battle/attack_calculator.gd")
    var GC = preload("res://resources/game_constants.gd")

    print("\n========== 伤害计算测试 ==========")

    # 创建攻击者（野战炮）
    var attacker = UnitStats.new()
    attacker.combat_kind = GC.CombatKind.SUPPORT
    attacker.attack_light = 135.0
    attacker.attack_armor = 90.0
    attacker.weapon_type = GC.WeaponType.INDIRECT

    # 创建防御者（轻装步兵）
    var defender = UnitStats.new()
    defender.combat_kind = GC.CombatKind.LIGHT
    defender.defense_light = 8.0
    defender.defense_armor = 5.0
    defender.defense_air = 3.0

    # 计算伤害
    var damage = AttackCalculator.calculate_damage(
        attacker, defender, 500.0, GC.WeaponType.INDIRECT, 0, []
    )

    print("攻击者: 77mm野战炮")
    print("  combat_kind: SUPPORT")
    print("  attack_light: 135")
    print("  weapon_type: INDIRECT")
    print("\n防御者: 轻装步兵")
    print("  combat_kind: LIGHT")
    print("  defense_light: 8 (防直射)")
    print("  defense_armor: 5 (防曲射)")
    print("  defense_air: 3 (防空)")
    print("\n伤害计算:")
    print("  get_attack_vs(LIGHT) = ", AttackCalculator.get_attack_vs(attacker, GC.CombatKind.LIGHT))
    print("  get_defense_vs(INDIRECT) = ", AttackCalculator.get_defense_vs(defender, GC.WeaponType.INDIRECT))
    print("  calculated_damage = ", damage)
    print("  expected_damage = ", 135.0 * 100.0 / (100.0 + 8.0))
    print("  差异 = ", absf(damage - 135.0 * 100.0 / (100.0 + 8.0)))

    print("========== 测试完成 ==========\n")
```

## 游戏内测试步骤

### 1. 编译并启动游戏
```
1. 关闭 Godot 编辑器
2. 重新打开项目
3. 点击 "运行项目" (F5)
```

### 2. 观察控制台输出
```
查找以下关键输出:
- ✅ 验证通过
- weapon_type = 1 (INDIRECT)
- get_defense_vs 返回正确的防御值
- calculated_damage ≈ 127.36
```

### 3. 实际战斗测试
```
1. 进入关卡
2. 部署 77mm野战炮
3. 放置敌人（近处和远处各一个）
4. 观察:
   - 野战炮优先攻击远处敌人（索敌验证）
   - 抛物线弹道显示正确
   - 爆炸特效播放
   - 敌人 HP 下降
   - 伤害数值合理（约 83-128）
```

## 如果测试失败

### 情况1: 编译错误
```
错误信息: "Parser Error: Unexpected identifier..."
解决: 检查是否有语法错误，查看具体行号
```

### 情况2: 验证失败
```
错误信息: "ww1_77mm 卡牌不存在"
解决: 检查 default_cards.gd 中是否有 ww1_77mm 定义
```

### 情况3: 伤害为0
```
可能原因:
1. weapon_type 错误（不是 INDIRECT）
2. 防御值过高（攻击 < 防御）
3. 武器槽位未初始化
```

### 情况4: 敌人不掉血
```
可能原因:
1. 曲射批处理防御计算未应用
2. take_damage 方法有问题
3. HP 显示未更新
```

## 调试技巧

### 1. 启用详细日志
在项目设置中启用：
```
Debug → Editor → Script → Debug → Print Script Times
```

### 2. 使用断点调试
在关键位置设置断点：
- `AttackCalculator.calculate_damage`
- `SimpleIndirectProjectileBatch._apply_hit`
- `Unit.take_damage`

### 3. 查看变量值
在断点处查看：
- `raw_dmg` - 原始伤害
- `def_val` - 防御值
- `final_primary_dmg` - 最终伤害
- `target.hp` - 目标当前 HP
