# 法则施放特效 AI 提示词（25组）

使用说明：
- 存放路径：`assets/effects/laws/`
- 25组法则施放特效，每条法则需施放动画 + 持续效果 + 目标指示器
- 画幅：1024x1024
- 4大家族色系：钢铁 Steel（银灰）、烈焰 Flame（橙红）、雷霆 Thunder（蓝白）、虚空 Void（暗紫）
- 每条法则是一组特效，AI一次生成一张概念图
- 统一 Negative prompt 适用于全部25条

## 背景设定融入说明

本文档每条提示词均融入《相位战争》核心世界观：

- **法则(Phase Law)** 是超空间节点(Hyperspace Node)的"规则修改权限"——相位师激活后可改变局部战场规则，本质是将观测者的意识、信念、意志转化为可量化的现实干涉能力(Reality Interference)。
- **超空间视觉特征**：深灰蓝虚空，流动光带，浮岛节点——所有特效的背景氛围必须呼应这一维度。
- **我方标识色**：绿色（相位师的意识能量可视化），所有正面效果的能量底色均为绿色。
- **暗黑文明**：不可知的宇宙存在，天空裂缝中的黑暗——作为负面/威胁暗示元素。
- **4大家族理念**：铁壁公司"秩序与防御"、诺瓦军武"进攻与进化"、螺旋侦察"信息与机动"、虚空研究所"凝视深渊，深渊回望"。

English prompt 中统一融入的关键词：phase consciousness（相位意识）、hyperspace node（超空间节点）、will-made-manifest（意志具象化）、reality interference（现实干涉）。
