> 本文件所有图标提示词均深度融入《相位战争》(Phase War) 核心世界观：
> - **超空间**：时间线交汇处的稳定间隙，深灰蓝虚空背景+流动光带+浮岛节点。
> - **7大势力**：超空间中最强大的7个相位师组织，每个势力拥有独特的相位信念与意识形态。

> - **资源**：纳米材料、合金、晶体、能量块——均为超空间中提取的相位物质。
