# 相位战争 — 我方载具精灵提示词（11 种）

> **背景设定融入说明：**
> 我方载具是相位师在超空间中通过"蓝图提取 + 纳米材料印制"生成的战斗单位。
> 这些不是传统工厂制造的军械，而是**科学与相位意识结合的具象化产物**。
> 视觉特征：纳米印制组装缝线（半透明全息蓝光）、蓝图残留纹理、相位能量核心。
> 我方标识：**绿色相位意识能量**（代表相位师的意志干涉力）。
> 画幅：1024x1024 或 1536x1536 竖图。存放路径：`assets/unit_sprites/`
