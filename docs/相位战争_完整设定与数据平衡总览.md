## 2. 核心系统架构

### 2.1 战斗系统

**胜负条件：**
- 胜利: 所有波次生成完毕 且 所有敌人被消灭
- 相位大师战: 摧毁敌方相位驱动核心
- 失败: 己方相位场驱动核心被摧毁

**部署规则：**
- 能量消耗 = 平台费用 + 武器费用之和，最低1
- 部署时间 = 费用 / 能量输出速率 (0.5s ~ 30s+)
- 单位上限: 5 (默认)，不可重复部署同平台
- 幽灵预览出现后计入上限

**战斗星级 (1-3星):**
| 星级 | 条件 |
|------|------|
| 1星 | 胜利即可 |
| 2星 | 单位存活率 >= 50% |
| 3星 | 存活率 >= 80% 且 用时 <= 预估时间的70% |

预估时间 = `wave_total * wave_interval + 15s`

**敌方波次：**
- 首波: 开战3秒后
- 间隔: 12s默认 (可配置)
- 波次规律: 第1波基础; 每3波(3,6,9...):精英; 最后一波:BOSS
- 敌人场上上限: 5

### 2.2 伤害公式

```
final_damage = base_damage * crit_multiplier * (1 - damage_reduction * (1 - armor_penetration))
splash = final_damage * splash_damage
shield_cap = 2 * max_hp
```

**暴击:** 2x倍率
**护盾上限:** 2倍最大生命值

## 7. 卡牌成长系统

### 7.1 成长路径总览

```
敌人掉落卡牌(1★ common)
    │
    ├──▶ 研究点升星 (1★ → 9★) ← 消耗研究点
    │       │
    │       ├──▶ 强化 (Lv1→Lv10) ← 消耗纳米材料
    │       │       │
    │       │       └──▶ 变异 (Lv5时有25%概率触发)
    │       │
    │       └──▶ 改装 (最多3次) ← 消耗研究点+许可函
    │               │
    │               ├── 4★ → 进化1阶 ← 消耗320研究点+许可函
    │               │       │
    │               │       └── 7★ → 进化2阶(阵营分支) ← 消耗680研究点+许可函
    │               │
    │               └── 附魔 (每升星获得1个词缀)
    │
    └──▶ 重复卡 → 分解为研究点
```

### 7.6 战力评分公式

```
power_score = (80 + star×28 + mod_count×22) × rarity_mul × (1 + inherit_bonus)
```

| 组成项 | 说明 |
|--------|------|
| 80 | 基础分 |
| star × 28 | 每星28分, 9星=252 |
| mod_count × 22 | 每次改装22分, 满改=66 |
| rarity_mul | 品质倍率(1.0~1.8) |
| inherit_bonus | 进化继承加成(0~0.9) |

**成长应用:**
```
HP *= avg(hp_mul across all components) × (1 + inherit_bonus) × rank_hp_mul
DMG *= avg(dmg_mul across all components) × (1 + inherit_bonus) × rank_dmg_mul
```
