## 一、现有属性系统回顾

### 1.1 步兵单位基础属性结构

```gdscript
# 以 ww1_mp18 为例
{
    "id": "ww1_mp18",
    "name": "MP18突击班",
    "combat_kind": 0,  # LIGHT - 轻装兵
    "power": 15,
    "max_hp": 100,
    "move_speed": 80,  # px/s
    "attack_range": 120,  # px
    "attack_interval": 1.0,  # s
    "deploy_speed": 4,  # 0最快, 7最慢
    "attack_light": 35,   # 对轻装攻击力
    "attack_armor": 0,    # 对装甲攻击力
    "attack_air": 0,      # 对空中攻击力
    "defense_light": 8,   # 对轻装防御
    "defense_armor": 5,   # 对装甲防御
    "defense_air": 3      # 对空中防御
}
```

## 二、步兵强化等级（Lv1-Lv10）— 真实军衔映射

每个等级对应真实军事称号，**名称独立于兵种**，适用于所有步兵单位：

| 等级 | 军事称号 | 现实依据 | 战力倍率 | 消耗倍率 |
|------|---------|---------|---------|---------|
| Lv1 | 征召兵 | 完成基础训练，配发标准装备 | 1.00× | — |
| Lv2 | 合格步兵 | 通过战术考核，熟悉班组配合 | 1.05× | 0.5× |
| Lv3 | 老兵 | 经历过实战，战场直觉形成 | 1.10× | 1.0× |
| Lv4 | 精锐 | 连队尖子，可担任火力组长 | 1.15× | 1.5× |
| Lv5 | 士官 | 获得军士军衔，领导一个火力组 | 1.20× | 2.0× |
| Lv6 | 战斗老兵 | 多次实战部署，全战术精通 | 1.25× | 2.5× |
| Lv7 | 三级军士长 | 可训练新兵，担任排士官长 | 1.30× | 3.0× |
| Lv8 | 二级军士长 | 营级特等射手/战斗教官 | 1.35× | 3.5× |

**强化效果**（已有系统）：
- 每级增加 5%（Lv1-8）/ 跳变（Lv9=50%, Lv10=60%）
- 对 combat_unit：HP + 防御类 + 重量
- 能量消耗微增：`energy_cost × (1.0 + bonus × 0.5)`
