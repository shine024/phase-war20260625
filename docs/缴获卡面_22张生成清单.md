- [ ] 视角是否为严格正侧面（非 3/4）？
- [ ] 是否为完整全身（从头到脚，包括腿和靴子）？
- [ ] 抠图边缘是否干净（无白边/杂色/锯齿）？
- [ ] 主体是否居中？
- [ ] 画风是否符合 C方案（写实概念艺术，非卡通）？
- [ ] 是否有禁止元素（人脸/现实军徽/文字/水印）？
- [ ] 整体质量是否达标（清晰、细节丰富）？
