# Nano Banana 2 敌人分帧精灵图提示词（36 套）

目标：每个敌人输出 **1 张精灵图集（sprite sheet）**，包含 3 段动画：**前进 / 进攻 / 死亡**。  
风格：严格 2D 正侧视 + 正交投影 + 白底。

---

## 统一规格（每条都已内置）

- 图像类型：**一张多帧 sprite sheet**（不是单张立绘）
- 动画分段：3 行布局
  - 第 1 行：前进（move）8 帧，循环
  - 第 2 行：进攻（attack）8 帧，循环
  - 第 3 行：死亡（death）8 帧，不循环，最后 1 帧为静止残骸/失效状态
- 帧规格：
  - 每帧等尺寸（建议 256x256）
  - 帧间距固定（建议 16px）
  - 单位比例一致（占帧高约 75%-85%）
  - 统一侧视朝向（面向右或面向左，整张一致）
- 背景：纯白（#FFFFFF）
- 禁止项：三分之四视角、透视、俯仰视角、等距、鸟瞰、场景背景、地面、文字、水印、logo、人物
