## 游戏简介

**相位战争** 是一款基于 Godot 4.x 的移动端策略卡牌塔防游戏。

核心概念：玩家扮演"相位师"，使用"相位仪"控制单位在战场上作战，通过能量管理、卡牌部署和技能施放来抵御敌人。游戏没有传统"等级"概念——角色强弱完全由**相位仪属性 + 能量场刻印 + 技能 + 装备**决定。

## 核心系统概览

### 1. 相位仪系统（替代等级）
- 13种标准属性（攻击、防御、HP、能量容量/恢复、暴击率/伤害、移速、攻速、掉落率等）
- 按星级(1-7)线性成长，同一星级内属性随机浮动
- 7种稀有度：Common → Uncommon → Rare → Epic → Legendary → Mythic → Unique
- 可掉落、商店购买、合成获取
- 稀有属性如"零成本部署"、"额外部署槽"等

### 2. 卡牌系统
- 60+张卡牌，分为：平台、武器、能量卡
- 5大时代主题：一战(蒸汽) → 二战(机械) → 冷战(导弹) → 现代(信息) → 近未来(相位科技)
- 合成系统：两张同卡 → 一张高星卡（80%/90%成功率）
- 词条系统：卡牌可携带2-3个随机词条
- 词条锁定符：10%概率获取，锁定合成时保留的词条

### 3. 能量场刻印系统（核心养成）
- 8种能量场（堑壕死、钢铁洪流、核烬、磁暴、量子虚空、炼狱、雷霆末世、虚空裂隙）
- 每种能量场有5-6个刻印词条模板，共40+个
- 刻印通过不同时代的关卡激活并刻录到相位仪上
- 进度机制：0.0~1.0，进度满后完整生效，可随时删除
- 刻印分5种稀有度（common→mythic），影响数值强度

### 4. 战斗系统
- 实时自动战斗 + 玩家能量部署
- 开局100能量，每秒约净回复0.5
- 同时部署5个单位上限（受相位仪影响可增加）
- 主动法则：50能量消耗，相位师主动技能
- 战斗波数：一战3-5波 → 近未来7-10波

### 5. 战力评估系统（无等级）
- 6维度加权评估：
  - A. 相位仪属性：15%（HP/攻击/防御/能量/单位上限各占子项）
  - B. 刻印词条：25%（刻印稀有度 × 进度 × 效果强度）
  - C. 特质：10%（特质效果强度评分）
  - D. 主动技能：15%（冷却/消耗/效果强度）
  - E. 被动技能：10%（效果类型+强度）
  - F. 装备槽位：25%（平台数×30 + 武器数×40 + 能量卡数×30）
- 7星制：1★(0-250) / 2★(250-600) / 3★(600-1200) / 4★(1200-2200) / 5★(2200-3800) / 6★(3800-6000) / 7★(6000+)

### 6. 相位师名册
- 50名相位师统一管理（20我方/中立 + 30敌方）
- 7大势力：铁壁要塞、赤刃会、阳光熔炉、虚空行者、灰烬教团、霜冠庭、中立
- 我方相位师各有独特特质和技能组合
- 敌方按难度分档：easy/medium/hard/elite/boss
- 排行榜显示50名：排名、名称、势力、战力

### 7. 养成循环
- 碎片累计不扣除的升星机制
- 100关主线 + 爬塔模式
- 相位师对战（15%随机遭遇）
- 7大势力贡献度系统

> 请根据以下7大评审领域给出详细、具体的建议：

### 评审领域
1. **数值平衡** — 能量经济、难度曲线、权重分配、星级间距
2. **系统深度** — 卡牌使用率、合成反馈、刻印策略深度、无等级设计的目标感
3. **循环设计** — 100关长度、爬塔联动、随机遭遇体验、刻印进度填充速度
4. **战斗体验** — 自动战斗互动性、主动法则机会、侦查单位取舍
5. **养成节奏** — 碎片正反馈、锁定符稀有度、势力系统存在感
6. **潜在问题** — 胜利条件单一性、掉落可控性、掉落vs商店平衡、刻印删除成本
7. **新系统评审** — 刻印与词条重叠、能量场分布、刻印进度获取方式、20名我方差异化
8. **敌方设计** — 敌我装备体系差异(附录A.5)、30名敌方相位师的难度递进合理性、Tier 3技能强度（复活/无敌/永久增益）
9. **关卡节奏** — 法则开放规则的渐进体验(2.4)、环境循环的可预测性(2.5)、100关总长度的玩家留存
