# 相位战争 (Phase War) - AI 评审用游戏设计文档

> 本文档由代码逆向整理，供 AI 详细审阅游戏机制并提出改进意见。
> 项目技术栈：Godot 4.5 / GDScript / Mobile

---

## 一、游戏概述

### 1.1 核心定位
相位战争是一款**横版塔防 + 卡牌养成 + Roguelike 爬塔**混合策略游戏。玩家通过收集和升级载具/武器卡牌，配置相位仪（装备系统），施放相位法则（战时魔法），在跨越5个时代（一战→近未来）的100关战役中推进，并可与AI"相位师"进行PvP对战。

### 1.2 核心循环
```
战前准备（配置相位仪/激活法则/消耗资源）
    → 进入战斗（实时自动战斗 + 能量部署 + 法则施放）
    → 战后结算（星级评定/碎片掉落/资源奖励）
    → 养成升级（蓝图升星/合成强化/词条洗练/相位仪进阶）
    → 下一关
```

### 1.3 游戏阶段
游戏分为三个大阶段（GamePhase 枚举）：
- **PRE_BATTLE**：战前准备，配置装备和法则
- **BATTLE**：战斗进行中
- **POST_BATTLE**：战后结算和奖励

## 二、关卡与世界设定

### 2.1 时代系统
共100关，分5个时代，每个时代20关：

| 时代 | 关卡范围 | 敌人波次范围 | 每波敌人数 | 波次间隔 | 掉落倍率 |
|------|---------|------------|----------|---------|---------|
| 一战 (WW1) | 1-20 | 3-5波 | 1-2人 | 14秒 | 0.85x |
| 二战 (WW2) | 21-40 | 4-7波 | 2-3人 | 13秒 | 1.0x |
| 冷战 (Cold War) | 41-60 | 5-8波 | 2-3人 | 12秒 | 1.15x |
| 现代 (Modern) | 61-80 | 6-9波 | 2-4人 | 12秒 | 1.25x |
| 近未来 (Near Future) | 81-100 | 7-10波 | 3-4人 | 11秒 | 1.4x |

波次数和每波敌人数在时代内线性插值（第1关=min，第20关=max）。

- **terrain**: plain / city / mountain / desert 等
- **energy_field**: normal / high_field / nano_fog 等
- **time_of_day**: day / night 等

## 四、战斗系统

### 4.1 战斗基本规则
- **视角**：横版，敌人从右侧进入
- **玩家单位上限**：5个
- **敌方单位上限**：5个
- **胜负条件**：
  - 胜利：摧毁所有敌方波次的所有敌人
  - 失败：玩家相位场驱动器被摧毁 / 玩家基地被突破
- **星级评定**：根据战斗统计数据计算（0-3星）

### 4.2 能量系统
- **能量上限**：默认100（由能量卡决定，1-7星对应100-700）
- **开局能量**：默认100（由能量卡决定）
- **能量回复**：每秒 = 基础回复(1.0/s) + 能量卡加成 - 相位仪消耗(0.5/s)
- **部署消耗**：载具卡费用即为部署能量消耗
- **战后补能**：自动消耗能量块（基础资源）补满

### 4.3 战斗流程
1. 进入战斗场景
2. 初始化空间分区系统（性能优化）
3. 波次刷新系统按时间间隔生成敌人
4. 玩家通过点击部署载具（消耗能量）
5. 实时自动战斗（索敌、攻击、移动）
6. 所有波次清完 → 胜利 / 基地被毁 → 失败
7. 结算：星级评定、掉落计算

### 4.4 伤害与掉落系统
- 敌人死亡时触发蓝图碎片掉落（基于敌人原型定义的掉落表）
- 掉落概率受关卡掉落倍率影响
- 侦查/隐匿单位（SCOUT/STEALTH）额外提升掉落数量和概率
- 法则碎片独立掉落：普通15%，精英30%，Boss 50%

### 4.5 相位师对战（PvP Boss）
- 每场战斗15%概率遭遇相位师（第49关固定遭遇）
- 优先遭遇当前关卡所属势力的相位师
- 相位师战斗禁用波次系统，使用特殊战斗配置
- 7位预设相位师：终焉之镰、炽焰星痕、雷霆判官、寒霜壁垒、量子幽灵、虚空低语、边境开拓者

### 4.6 敌人类型
- **普通敌人**：4种行为标签（infantry/frontline, infantry/backline, turret/sustained, artillery/backline）
- **精英敌人**：更强属性，提高掉落
- **Boss敌人**：最高属性，高掉落
- **蜂群单位**：轻量化路径（无刚体血条、MultiMesh绘制），用于大规模低级敌人
- 按时代分类：每个时代约7种基础+精英+Boss敌人

## 六、相位法则系统（战争魔法）

### 6.1 法则家族
4个法则家族，每个家族有被动和主动法则：

| 家族 | 中文名 | 主题 |
|------|--------|------|
| STEEL | 钢铁 | 防御/护甲/修复 |
| FLAME | 烈焰 | 火焰/燃烧/爆破 |
| THUNDER | 雷霆 | 电击/EMP/链式攻击 |
| VOID | 虚空 | 时间/空间/熵 |

### 6.2 法则类型
- **被动法则**：战前激活，整场战斗持续生效
  - 消耗纳米材料激活
  - 需要碎片解锁
  - 有环境条件要求
- **主动法则**：战斗中施放
  - 消耗能量 + 可选纳米材料
  - 有施放条件（如最少友方单位数）
  - 有每场战斗施放次数限制

### 6.3 法则数据结构
每条法则包含：
- `shard_req`：解锁所需碎片数量（5-20+）
- `research_req`：知识研究需求（defense/energy/mobility/mystic knowledge）
- `env_req`：环境条件要求（地形/天气/能量场/时间）
- `activate_cost`：战前激活消耗（纳米材料）
- `battle_cost`：战中施放消耗（能量 + 纳米）
- `cast_conditions`：施放条件（最少友方单位/每场次数限制）
- `env_changes`：施放后对环境的修改
- `runtime_tags`：运行时效果参数（效果类型/半径/持续时间/数值/缩放/目标）

### 6.4 法则示例
- **钢铁·相位装甲**（被动）：载具护甲+15%，需要城市/山地地形，5碎片，20纳米
- **烈焰·热能过载**（被动）：攻击附带燃烧4点/3秒，需要高能量场/纳米雾，5碎片，25纳米
- **雷霆·电磁风暴**（主动）：AOE电磁脉冲，半径380，持续6秒，需2友方单位，50能量，每场2次

### 6.5 法则碎片系统
- 碎片通过击杀敌人获得（普通15%，精英30%，Boss 50%）
- 碎片类型根据当前关卡环境匹配
- Boss掉落3碎片，精英2碎片，普通1碎片
- 侦查单位额外提升掉落数量
- 新游戏赠送50碎片 + 4条初始法则

### 16.3 刻印机制
- **刻印槽位**：由相位仪稀有度决定槽位数
- **刻印进度**：每条刻印有进度（0.0~1.0），进度满后词条完整生效
- **可删除**：可随时删除刻印，进度数据保留，重新激活不丢进度
- **我方敌方共用**：同一套刻印规则，敌方相位师也有刻印

### 16.4 刻印词条类型（40+条，按效果分类）

**防御类示例：**
- 死盾铭刻：最大生命+5%~25%（按进度）
- 钢铁意志：防御+3%~15%
- 纳米修复：每秒回复 0.5%~3% 最大生命

**攻击类示例：**
- 核能蓄积：攻击+4%~20%
- 雷霆一击：暴击率+2%~10%
- 深渊侵蚀：攻击附带 5%~15% 最大生命真实伤害

**能量类示例：**
- 量子共鸣：能量上限+10%~50%
- 时间加速：能量回复+5%~25%
- 深渊吞噬：击杀敌人回复 10~50 能量

**特殊类示例：**
- 虫群心智：单位上限+1~3
- 时间回溯：单位死亡后30%概率重生（冷却30秒）
- 纳米适应：战斗中每10秒随机获得一项临时增益

### 16.5 刻印来源
- 通过对应能量场的关卡有概率解锁可刻印词条
- 稀有刻印仅在高难度关卡出现
- Boss关卡固定解锁一条该能量场的稀有刻印

## 十七、相位师战斗力评估系统

### 17.1 核心设计理念
- 相位师**没有"等级"概念**，能力完全由相位仪 + 刻印 + 技能决定
- HP/攻击/防御/能量/单位上限全部来自相位仪本身
- 战力评估采用6维加权模型，总分映射为1★~7★

### 17.2 评估维度

| 维度 | 权重 | 说明 |
|------|------|------|
| A. 相位仪基础属性 | 15% | 仪器的HP/ATK/DEF/Energy/UnitLimit |
| B. 刻印词条 | 25% | 已刻录的进化词条（含进度） |
| C. 特质强度 | 10% | 相位师固有特质 |
| D. 主动技能 | 15% | 爆发与实用技能 |
| E. 被动技能 | 10% | 持续战斗优势 |
| F. 装备槽位 | 25% | 平台/武器/能量卡种类丰富度 |

### 17.3 星级分界

| 星级 | 名称 | 分数范围 | 颜色 |
|------|------|---------|------|
| 1★ | 新锐 | 0~250 | 蓝 |
| 2★ | 精英 | 250~600 | 绿 |
| 3★ | 高手 | 600~1200 | 黄 |
| 4★ | 大师 | 1200~2200 | 橙 |
| 5★ | 宗师 | 2200~3800 | 红 |
| 6★ | 传说 | 3800~6000 | 紫 |
| 7★ | 神话 | 6000+ | 金 |

### 17.4 各维度评分方法

**A. 相位仪属性（15%）：**
- 以中位数仪器数据为基准（HP=1200, ATK=40, DEF=55, EnergyCap=200, EnergyRegen=2.2, UnitLimit=7）
- 各属性内部加权（HP=25%, ATK=20%, DEF=20%, EnergyCap=10%, EnergyRegen=10%, UnitLimit=15%）
- 归一化到0~100后乘以权重

**B. 刻印词条（25%）：**
- 按刻印稀有度给基础分（Common=5, Uncommon=10, Rare=18, Epic=28, Legendary=40）
- 乘以进度值（0.0~1.0）
- 上限为已安装的刻印槽数 × 最高分
- 按已使用槽位比例填充到上限

**C. 特质（10%）：**
- 按特质稀有度评分（Common=8, Uncommon=15, Rare=25, Epic=38, Legendary=50）
- 多特质取加权平均（稀有特质权重更高）

**D. 主动技能（15%）：**
- 按技能强度等级评分（Tier1=10, Tier2=20, Tier3=35, Tier4=50, Tier5=70）
- 多技能取最高值×0.6 + 次高值×0.3 + 第三×0.1

**E. 被动技能（10%）：**
- 与主动技能类似的分级评分
- 取所有被动技能的平均值

**F. 装备槽位（25%）：**
- 计算已装备的独立平台/武器/能量卡种类数
- 平台种类分：已使用/总种类(23种) → 0~40分
- 武器种类分：已使用/总种类(27种) → 0~40分
- 能量卡种类分：已使用/总种类(10种) → 0~20分
- 总和即为装备维度分数

### 17.5 排行榜显示
- 显示50名相位师：排名、名称、当前所属势力、综合战力、星级
- 我方与敌方共用同一套评估规则
- 排行榜用于游戏内的相位师PvP匹配和展示

### B.4 敌方相位师技能设计规律

**主动技能特点：**
- Tier 1：单目标/小范围，冷却15秒，消耗60-80能量
- Tier 2：中范围AOE/召唤，冷却12秒，消耗80-120能量
- Tier 3：大范围/多目标，冷却10秒，消耗100-150能量，附带控制效果

**被动技能特点：**
- Tier 1：单体属性增益（防御/攻击/速度+10-15%）
- Tier 2：范围增益+条件触发（死亡护盾/击杀回能）
- Tier 3：强力光环+永久增益+复活/无敌等特殊机制

**装备配置规律：**
- Tier 1：基础等级装备，2个平台+2种武器
- Tier 2：进阶等级装备，2个平台+2-3种武器
- Tier 3：专家等级装备，2-3个平台+3-4种武器，后期使用混合/神级相位仪

### B.5 遭遇规则

- 每场战斗15%概率遭遇相位师
- 第49关固定遭遇相位师
- 优先匹配当前关卡所属势力的相位师
- 相位师战斗禁用波次系统，使用特殊战斗配置
- 难度匹配：关卡难度越高，越可能遭遇高Tier相位师
