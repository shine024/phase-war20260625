**位置**: `managers/game_manager.gd`

**修改前**:
```gdscript

```

**修改后**:
- ✅ 完整的星级计算算法
- ✅ 多维度评分系统
- ✅ 动态星级（1-3星）

**评分标准**:
1. **剩余能量** (30%权重)
   - 剩余50%以上能量 = 满分

2. **损失单位** (40%权重)
   - 损失少于20% = 满分
   - 损失越少分数越高

3. **战斗时间** (30%权重)
   - 理想时间: 60-180秒
   - 太快或太慢都会扣分

**星级标准**:
- 3星 (80分以上): 完美胜利
- 2星 (50-80分): 普通胜利
- 1星 (50分以下): 艰难胜利
