### 2. 成就面板布局修复 ✅

- 改用居中布局 (anchors_preset = 8)

**修复前**:
```gdscript

```

**修复后**:
```gdscript

offset_left = -300.0
offset_top = -250.0
offset_right = 300.0
offset_bottom = 250.0
```
